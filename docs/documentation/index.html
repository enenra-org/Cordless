Commands

Main

Prefix

ADMIN ONLY

The prefix is by default "&" and can be changed with the following command. 
&pre-change <new prefix here>

Help

This command lists out all th ecommands and what they do. 
&help

Memes

Memes

This command returns one meme. 
&meme

Memeflow

This command starts/stops a flow of memes every 10 seconds. 
&startflow
&stopflow

Music

Play

This command retrieves the input after it as a YouTube video and plays it in your current channel. 
&play <song name here>

Pause

This command pauses the current song queue. 
&pause

Resume

This command resumes a paused song queue. 
&resume

Queue

This command outputs the full queue of songs that are going to play on your server. 
&queue

Skip

This command skips the currently playing song in your server, or skips more. 
&skip <number of songs to skip, defaults to 1>

Loop

This command loops the current queue. 
&loop

Volume

This command sets the volume of the music. 
&setvol <a number between 0-200>

Search

This command searches youtube for your keyword for a song to play. 
&search <youtube search keywords>

Clear Queue

This command clears the current song queue. 
&clearqueue

Show Current Song

This command shows the currently playing song. 
&np

Leave Voice Channel

ADMIN ONLY

This command forces Cordless to leave the voice channel. 
&leavemus
